Developing a Website

To make a basic website, you'll need to know some HTML and CSS.

HTML

HTML defines the structure and contents of a web page. It is where things go, how they are laid out, and what's on the page.

The main component of an HTML structure is an HTML tag. For example, <p>...</p> is a paragraph of text. Here is the simplest document structure for any HTML page:

Note: the first/main page of every website you create must be saved as index.html for it to be recognized.

Explaining the Code

!DOCTYPE html is the initial declaration of the document.

html lang='en' states that the code in your file is written in English.

head marks the start of the head section. Most of what is in here will not be shown on the screen, it deals more with what is going on under the hood.

meta charset='utf-8' defines the character set that is used to write the document, just use this as is.

<title>Hello, World!</title> is the title of the page and what users will see in the title bar of their browsers.

body marks the start of the body section. It's the main part of the document and where all of the content of the page goes. Anything that you would like to appear on the page goes here.

<h1>Hello, World!<h1> is the main header on the page.

<p>My first web page.</p> is a paragraph of text.

</html> is the closing tag for the entire HTML document. You have to include this at the end for your file to run properly.

Viewing Your Website

To view your freshly-developed website, be sure to save the file. Then, navigate to the file in your file explorer/finder, and double click on the file index.html. Congrats, you've set up your first web page! But...it's kind of ugly, isn't it? Let's get into CSS then.
