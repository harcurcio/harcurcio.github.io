Welcome to How2Start!

This website is created for people who don't really know how to code, and the idea of blindly starting a project overwhelms them to no end. I've been there too, it's a horrible feeling with seemingly no end.

So, I took it upon myself to create a website that teaches you how to start some of the most common coding projects out there. It's not the most detailed, but it's full of supplemental learning resources. Get out there and learn some code! Good luck!
